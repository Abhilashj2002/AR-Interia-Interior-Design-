# Run and deploy your Design Studio app

This contains everything you need to run your app locally.

View your app in Design Studio: https://ai.studio/apps/drive/1SlaF0-q5nJsjmREa7nDowSaW1V2GSUdR

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_PRO_ENGINE_KEY` in [.env.local](.env.local) to your Smart Engine API key
3. Run the app:
   `npm run dev`
