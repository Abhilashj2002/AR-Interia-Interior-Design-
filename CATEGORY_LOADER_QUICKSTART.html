📋 Category Format

JSON Format

{ "id": "cat-living-room", "title": "Living Room", "description": "Modern designs", "icon": "🛋️", "color": "#3B82F6", "image": "https://url.jpg", "status": "active" }

CSV Format

Title,Description,Icon,Color,Image,Status Living Room,Modern designs,🛋️,#3B82F6,https://url.jpg,active Bedroom,Luxury designs,🛏️,#EC4899,https://url2.jpg,active

Required Fields

  • title - Category name (required)
  • description - Category description (required)
  • icon - Emoji icon (optional, e.g., 🛋️)
  • color - Hex color code (optional, e.g., #3B82F6)
  • image - Image URL (optional)
  • status - "active" or "inactive" (optional, default: active)

🚀 Getting Started

Step 1: Download Sample File

Open the Category Loader and download either JSON or CSV sample file

Step 2: Edit the File

Open in Excel (CSV) or any text editor (JSON) and add your categories

Step 3: Upload File

Use the Category Loader UI to upload your edited file

Step 4: Preview & Submit

Review the preview and click "Submit to Backend" to save to database

✅ Your categories will now appear on the website!

🔌 Backend Integration

The categories are automatically submitted to the backend API. The endpoint expects:

POST http://localhost:5174/api/categories Request Body: { "categories": [ { "title": "Living Room", "description": "Modern designs", "icon": "🛋️", "color": "#3B82F6", "image": "https://image.jpg", "status": "active" } ] }
